*Скрипт написан в рамках тестового задания для соискания должности "программист-секретарь"*

Данный скрипт позволяет скачивать и загружать файлы из GoogleDrive с помощью Google Drive API.

Все необходимые для написания пакеты устанавливались в виртуальное окружение.
Список использованных пакетов и их версии находится в файле requirements.txt.

Поскольку Google не позволяет напрямую через скрипт получать доступ к даным чужого аккаунта,
авторизация пользователя через скрипт с помощью login/password не была реализована.
При запуске скрипта необходимо вручную перейти по предоставляемой ссылке и выполнить вход в
необходимый аккуант google, только после этого будут доступны скачивание и загрузка файлов.

Вызов утилиты осуществляется так:

drive_api.py put/get src_path dst_path

Если вызван метод get, то файл, находящийся на сервере под именем src_path, скачивается и
сохраняется по локальному пути dst_path. Путь назначения (dst_path) должен обязательно
существовать, а также у пользователя должны быть права записи в данной директории.

Если вызван метод put, соответствующий файл src_path загружается на сервер по пути dst_path.
Путь назначения (dst_path) предполагает такой вид: folder/file_name или же просто file_name.
Если папка folder на диске не существует, она бдет создана.
!!! Уровень вложенности файла на google диске предполагает максимум одну папку !!!
